Nōva

Tu workspace mínimo. Notas, imágenes y tareas organizados en un solo lugar, con sincronización en tiempo real y cero dependencias.

Ingresar a la app

Notas

Editor de texto rico con H1, H2, negrita, cursiva, subrayado, listas, checkbox y alineación. Layout masonry con tarjetas que se auto-guardan. Múltiples instancias para mantener todo separado.

Imágenes

Galería con grid masonry, vista previa en modal a tamaño completo y edición de títulos. Agregá imágenes por URL al instante.

To-do

Tareas con edición inline, prioridades con color, fechas límite, drag & drop para reordenar y barra de progreso. Vista de tabla y kanban intercambiables.

Hecho con vanilla JS

Sin frameworks, sin bundler, sin dependencias. HTML, CSS y JavaScript puro con ES Modules importados directamente desde CDN. Ligero, rápido y fácil de entender.

HTML5 CSS3 ES Modules

Sync en tiempo real

Powered por Firebase Auth y Firestore. Tus datos se sincronizan instantáneamente. El indicador de sync te da feedback visual cada vez que se escribe en la base de datos.

Firebase Auth Firestore onSnapshot

Diseño pensado

Dark mode por defecto con toggle a light mode. Tipografía Space Grotesk + Inter. Grid de puntos sutil, bordes neón y transiciones suaves. Responsive para desktop y mobile.

Dark/Light Responsive CSS Vars

To-do completo

Prioridades con indicador de color (rojo, amarillo, verde), fechas límite con aviso de vencimiento, edición de texto inline con doble click, reordenado por arrastre en la tabla y barra de progreso que muestra tareas completadas en tiempo real.
